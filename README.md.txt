# Event Landing Page with TikTok Pixel

هذه صفحة هبوط بسيطة للتواصل عبر **واتساب**، مضاف لها كود **TikTok Pixel** لتتبع الأحداث.

## المحتويات
- `index.html` : الصفحة الرئيسية.
- `assets/css/style.css` : التنسيقات.
- `assets/js/script.js` : سكربت تتبع النقرات وعرض الصفحة.

## كيفية التشغيل
1. افتح `index.html` في المتصفح.
2. أو ارفع المشروع على **GitHub Pages** أو أي استضافة مثل **Netlify**.

## المميزات
- تصميم متجاوب وبسيط.
- زر واتساب مع تتبع أحداث.
- تكامل مع TikTok Pixel.
